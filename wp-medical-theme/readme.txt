== Description ==

Современная тема для медицинской клиники с адаптивным дизайном, полностью скопированная с оригинального сайта.

== Installation ==

1. Загрузите папку `wp-medical-theme` в директорию `/wp-content/themes/` вашего WordPress сайта
2. Активируйте тему через админку WordPress (Внешний вид → Темы)
3. Установите необходимые плагины (если требуется)

== Features ==

* Адаптивный дизайн
* Custom Post Types для врачей и услуг
* Шорткоды для динамического контента
* Настройка контактной информации через Customizer
* Поддержка мультиязычности
