# Как подключить базу данных Supabase

## Вариант 1: Самый простой способ

### Шаг 1: Получите данные для подключения
1. Перейдите в панель управления Supabase
2. Выберите ваш проект
3. Слева в меню нажмите "Project Settings" (шестеренка)
4. Выберите "API"
5. Скопируйте:
   - Project URL (пример: `https://abcdefghijklmnopqrst.supabase.co`)
   - anon public key (длинная строка)

### Шаг 2: Добавьте переменные в Netlify
1. Перейдите в настройки сайта в Netlify
2. Site settings → Build & deploy → Environment
3. Добавьте:
   ```
   SUPABASE_URL = ваш_project_url
   SUPABASE_KEY = ваш_anon_key
   ```

### Шаг 3: Пересоберите сайт
1. В Netlify нажмите "Deploys"
2. Нажмите "Trigger deploy" → "Clear cache and deploy site"

## Вариант 2: Если вы используете код

### Файл подключения: `netlify/functions/simple-supabase-client.js`
```javascript
const { createClient } = require('@supabase/supabase-js');

const supabaseUrl = process.env.SUPABASE_URL;
const supabaseKey = process.env.SUPABASE_KEY;

if (!supabaseUrl || !supabaseKey) {
  throw new Error('Не заданы переменные окружения SUPABASE_URL или SUPABASE_KEY');
}

const supabase = createClient(supabaseUrl, supabaseKey);
module.exports = supabase;
```

### Использование в функциях:
```javascript
const supabase = require('./simple-supabase-client');

// Загрузка данных
const { data, error } = await supabase
  .from('doctors')
  .select('*');

// Сохранение данных
const { error } = await supabase
  .from('doctors')
  .insert([
    {
      name: 'Иван Петров',
      specialty: 'Кардиолог'
    }
  ]);
```

## Готово!

После этого ваш сайт будет использовать Supabase для хранения данных.
